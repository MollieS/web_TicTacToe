# Tic Tac Toe : Web

Web version of Tic Tac Toe

Deployed to heroku at https://javattt.herokuapp.com/

## Requirements

* activator 1.3.10
* play 2.5.4 "streamy"
* sbt 0.13.11

## To run locally:

`git clone` this repository
`cd web_TicTacToe`
`activator run` to serve at localhost:9000

## To test:

From the root of the repository run `activator test`
